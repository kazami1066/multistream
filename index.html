Features

必要な機能が
すべて揃っています

シンプルな操作で、複数の配信を思い通りにレイアウト。

📺

最大9画面同時表示

1画面から最大9画面まで自由に増減。お気に入りの配信者を一度に並べて視聴できます。

↔️

ドラッグ&ドロップ

パネルをつかんでそのまま移動するだけ。自分好みのレイアウトに素早く並び替えられます。

🎛️

個別の再生コントロール

各パネルごとに再生・停止・ミュートを独立して操作。音量管理も思いのまま。

✏️

カスタムタイトル

各パネルの表示名を自分でつけ直せます。誰の配信かひと目でわかるよう整理できます。

🔍

追加前にサムネイル確認

URLを入力して「確認する」を押すとサムネイルとタイトルを表示。間違いなく追加できます。

🆓

完全無料・登録不要

アカウント作成もアプリのインストールも不要。ブラウザだけですぐに使い始められます。

How to use

3ステップで始められる

難しい設定は一切なし。開いてすぐ使えます。

1

プラットフォームを選んでURLを入力

YouTubeかTwitchを選び、見たい配信・動画のURLまたはユーザー名を入力。「確認する」でサムネイルを確認してから追加できます。

2

画面を好きなだけ追加・並び替え

右上の「＋」で画面数を増やし、複数の配信を追加。パネルをドラッグ&ドロップで自由に並び替えられます。

3

複数の配信を同時に楽しむ

各パネルの再生・停止・ミュートを個別に操作しながら、複数の配信をまとめて視聴できます。
